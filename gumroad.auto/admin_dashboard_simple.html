📋 How to Use (Server-Free Version)

  1. Create products first: Run python main.py in terminal
  2. Load products: Click "Load Products" button below and select the products/ directory
  3. Review products: Check each product's details
  4. Approve/Reject with feedback:
    • Approve: Select a reason, add notes, and mark as approved
    • Reject: Select a reason, add notes, and mark as rejected
    • Learning: Your feedback helps the system improve future products!
