# Gumroad商品ページ最適化テンプレート

## 説明文の構造（購買率を最大化する順序）

```markdown
[1行目: キャッチフレーズ（最重要・最初の2行に検索キーワード含める）]
**「[顧客の悩み]」を解決します。**

[なぜ今すぐ買うべきか：3つのメリット]
✅ [メリット1（数値や具体性があると良い）]
✅ [メリット2]
✅ [メリット3]

**[商品の内容]**
[アイコン] **[機能/章/スクリプト名]**（[ファイル名]）
[簡潔な説明・何ができるか1〜2行]

**こんな人に最適**
- [ペルソナ1]
- [ペルソナ2]
- [ペルソナ3]

[購入後の行動・ダウンロード方法]
購入後すぐに[形式]をダウンロードできます。
```

## タイトルの最適化ルール

1. **キーワードを前半に**: 「ChatGPT プロンプト集」ではなく「ChatGPTプロンプト集 -」
2. **具体的な数値**: 「プロンプト集」より「プロンプト集50選」
3. **ベネフィット**: 「仕事効率化」「コピペで使える」「即導入可能」
4. **最大文字数**: 80文字以内（Gumroadのサムネイルに収まるように）

**良い例**: `ChatGPTプロンプト集 - 仕事効率化50選【コピペ即使用】`
**悪い例**: `プロンプトをまとめたもの`

## タグ戦略（最大10個）

- 日本語タグ（日本市場向け）: 3〜4個
- 英語タグ（グローバル検索対応）: 3〜4個
- カテゴリ系: 1〜2個

**例（プロンプト集）**:
`ChatGPT, プロンプト, AI活用, 仕事効率化, prompt, productivity, template, Japan`

## 無料プレビュー戦略

- 全体の20〜30%を無料公開（`$0+`設定）
- 無料→有料へのアップセル導線を明記
- 例: 「全50プロンプトのうち10個を無料公開中。残り40個は¥980」
