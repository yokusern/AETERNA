# ChatGPTプロンプト集 - 仕事効率化50選

**対象**: ビジネスパーソン・フリーランス・エンジニア  
**価格**: ¥980  
**形式**: PDF（このMarkdownをPDFに変換）

---

## はじめに

このプロンプト集は、毎日の仕事でChatGPTを最大限に活用するための厳選50プロンプトです。  
コピー＆ペーストでそのまま使えます。必要に応じて`【】`内を書き換えてください。

## カテゴリ1: メール・文書作成（10プロンプト）

### 1. ビジネスメール作成
```
以下の状況でビジネスメールを書いてください。
状況: 【状況を説明】
宛先: 【相手の立場・関係性】
目的: 【このメールで達成したいこと】
トーン: 丁寧・簡潔に
件名も含めて作成してください。
```

### 2. メール返信の下書き
```
以下のメールへの返信を書いてください。
受信したメール: """
【受信メールの本文をここに貼り付け】
"""
私の立場: 【あなたの立場・役職】
返信の方向性: 【承諾/断る/確認事項あり など】
```

### 3. 議事録の整形
```
以下のメモから議事録を作成してください。
メモ: """
【会議メモをここに貼り付け】
"""
形式:
- 日時・参加者
- 決定事項（箇条書き）
- アクションアイテム（担当者・期限付き）
- 次回予定
```

### 4. 報告書の構成作成
```
【テーマ】について【対象読者（上司/クライアントなど）】向けの報告書を作成してください。
含める内容: 現状分析、問題点、提案、期待効果
分量: A4用紙【X】枚程度
専門用語は【多用する/なるべく使わない】
```

### 5. 提案書のエグゼクティブサマリー
```
以下の提案書の内容を、意思決定者が30秒で理解できるエグゼクティブサマリー（200字以内）に要約してください。
提案書の要点: 【主要な提案内容】
期待する効果: 【ROIや改善指標】
```

### 6. クレーム対応メール
```
クレームへの誠実な対応メールを書いてください。
クレーム内容: 【クレームの詳細】
事実関係: 【確認できている事実】
対応方針: 【具体的にどう対処するか】
謝罪は適切に行い、再発防止策を示してください。
```

### 7. 社内向けお知らせ文
```
社内向けお知らせを作成してください。
内容: 【お知らせする内容】
対象者: 【全社員/〇〇部門など】
実施日・締め切り: 【日付】
注意事項: 【あれば記載】
読みやすく、必要な行動が明確になるように書いてください。
```

### 8. プレゼン資料のスライド構成
```
【テーマ】についてのプレゼン資料の構成を作ってください。
発表時間: 【X分】
聴衆: 【役職・知識レベル】
目的: 【理解させたい/承認を得たい/行動を促したい】
各スライドのタイトルとキーメッセージ（1行）を提示してください。
```

### 9. 長文の要約
```
以下の文章を【100/200/300】字以内に要約してください。
重要なポイントを落とさず、専門用語は平易な表現に置き換えてください。

文章: """
【要約したいテキスト】
"""
```

### 10. 翻訳＋自然な日本語化
```
以下の英文を日本語に翻訳してください。
直訳ではなく、日本のビジネス文書として自然な表現にしてください。
専門用語は日本語の一般的な表現に合わせてください。

英文: """
【英文テキスト】
"""
```
