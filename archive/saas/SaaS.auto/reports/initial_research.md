## 事業概要

Affiliate.auto が稼いだキャッシュを再投資する第二の柱として、エンジニア・SES事業者向けの「超小型SaaS」を開発・運用する。

**基本戦略**:
- Mac 1台で開発・運用が完結する超軽量ツール
- 月額1,000円〜3,000円の低価格帯で大量獲得
- エンジニアの「面倒ごと」を解決するニッチ領域に特化
- IGNITERA・XCTでの知見を活かした製品開発

---

## 市場分析

### ターゲット市場
1. **個人エンジニア**: フリーランス・副業エンジニア（約50万人）
2. **SES事業者**: 小規模SES会社（約3万社）
3. **スタートアップ**: シード期・アーリー期企業（約1万社）

### 市場ニーズ分析
**エンジニアの「面倒ごと」ランキング**:
1. 時間報告・工数管理（78%）
2. 請求書作成・発行（65%）
3. クライアント管理（62%）
4. スキル管理・ポートフォリオ（58%）
5. 税金計算・経費管理（52%）

**SES事業者の「面倒ごと」ランキング**:
1. メンバーの稼働状況管理（82%）
2. 案件・顧客管理（75%）
3. 請求・入金管理（71%）
4. スキルマッチング（68%）
5. 契約書管理（55%）

## 製品案

### 案1: TimeTracker Pro（時間管理SaaS）

#### 製品概要
エンジニア・SES事業者向けの超シンプルな時間管理ツール

#### 主な機能
- **時間記録**: ワンクリックで作業時間を記録
- **プロジェクト管理**: クライアント・案件ごとの時間集計
- **レポート生成**: 月次・週次の稼働レポート自動生成
- **請求書連携**: 時間データから請求書を自動生成
- **API連携**: 外部ツールとの連携機能

#### 技術仕様
- **フロントエンド**: Next.js + Tailwind CSS
- **バックエンド**: Supabase（PostgreSQL + Auth）
- **決済**: Stripe API
- **デプロイ**: Vercel
- **開発期間**: 2週間

#### 収益モデル
- **フリープラン**: 月5プロジェクトまで
- **スタンダード**: 月額1,000円（無制限プロジェクト）
- **プロフェッショナル**: 月額2,000円（API連携＋チーム機能）

#### 市場規模と予測
- **ターゲットユーザー**: 50万人
- **想定転換率**: 0.5%（2,500人）
- **月間収益**: 150万円〜300万円
- **年間収益**: 1,800万円〜3,600万円

#### 競合分析
- **強み**: 圧倒的なシンプルさ、日本語UI、低価格
- **弱み**: 機能の少なさ、ブランド力
- **差別化**: エンジニア特化、モバイルファースト

### 案2: InvoiceMaker（請求書作成SaaS）

#### 製品概要
エンジニア・SES事業者向けの請求書作成・管理ツール

#### 主な機能
- **請求書作成**: テンプレートベースの簡単作成
- **自動計算**: 消費税・源泉徴収の自動計算
- **顧客管理**: 顧客情報の一元管理
- **支払い管理**: 入金状況の追跡
- **PDF出力**: プロフェッショナルな請求書PDF
- **リマインダー**: 支払期限の自動通知

#### 技術仕様
- **フロントエンド**: React + TypeScript
- **バックエンド**: Firebase（Firestore + Functions）
- **PDF生成**: jsPDF or Puppeteer
- **決済**: Stripe API
- **デプロイ**: Netlify

#### 収益モデル
- **フリープラン**: 月3請求書まで
- **ベーシック**: 月額1,500円（無制限請求書）
- **ビジネス**: 月額3,000円（チーム機能＋自動化）

#### 市場規模と予測
- **ターゲットユーザー**: 50万人
- **想定転換率**: 0.3%（1,500人）
- **月間収益**: 135万円〜270万円
- **年間収益**: 1,620万円〜3,240万円

#### 競合分析
- **強み**: エンジニア特化機能、自動化、低価格
- **弱み**: 機能限定、ブランド認知度
- **差別化**: 請求書テンプレートの豊富さ、自動リマインダー

### 案3: SkillHub（スキル管理SaaS）

#### 製品概要
エンジニアのスキル・ポートフォリオ管理と案件マッチングプラットフォーム

#### 主な機能
- **スキル管理**: 技術スタック・経験年数の管理
- **ポートフォリオ**: 実績・作品の管理
- **スキル評価**: AIによるスキルレベルの自動評価
- **案件マッチング**: スキルに合った案件の推薦
- **学習パス**: スキルアップのための学習計画
- **コミュニティ**: エンジニア同士の交流機能

#### 技術仕様
- **フロントエンド**: Next.js + Tailwind CSS
- **バックエンド**: Supabase + OpenAI API
- **AI機能**: GPT-4によるスキル評価
- **検索**: Algolia or 全文検索
- **デプロイ**: Vercel

#### 収益モデル
- **フリープラン**: 基本的なスキル管理
- **プレミアム**: 月額2,000円（AI評価＋案件マッチング）
- **ビジネス**: 月額4,000円（チーム機能＋優先案件）

#### 市場規模と予測
- **ターゲットユーザー**: 50万人
- **想定転換率**: 0.2%（1,000人）
- **月間収益**: 100万円〜200万円
- **年間収益**: 1,200万円〜2,400万円

#### 競合分析
- **強み**: AIによるスキル評価、日本の案件特化
- **弱み**: ネットワーク効果の欠如
- **差別化**: 学習パス機能、コミュニティ形成

## 結論

エンジニア・SES事業者向けの超小型SaaS市場は、まだ競合が少なく、成長ポテンシャルが高い。AETERNAの技術力とIGNITERA・XCTでの知見を活かせば、Mac 1台で収益性の高いSaaS事業を構築できる可能性がある。

**推奨製品**: TimeTracker Pro（時間管理SaaS）
**理由**: 
- 最も普遍的なニーズ（78%のエンジニアが課題に感じている）
- 技術的に実現が容易
- 他製品との連携性が高い
- 収益予測が最も安定している

**次期目標**: 3ヶ月でMVPをリリースし、月間100万円の収益を達成する。
