## 2. 課題と問題点（クリティカルなものから）

### 2.1 致命的な課題

#### 🔴 課題1: Tailwind CSS カスタムクラスが未定義

**場所**: `Affiliate.auto/production_site/app/globals.css`  
**問題**: globals.cssで `.glass-card`、`.gradient-btn`、`.mesh-bg` などのカスタムクラスを定義しているが、これらのクラス内で使用されている `shadow-card`、`shadow-card-hover`、`brand-*`、`surface-*`、`accent-*` などの色がTailwind configで定義されているにも関わらず、**一部クラス名が実際のTailwindの規則に従っていない**。

**影響**: サイトのデザインが崩れる、表示が正しくない

---

#### 🔴 課題2: アフィリエイトリンクが仮のもの

**場所**: `Affiliate.auto/production_site/app/page.tsx` (line 25, 40, 55, 70, 85, 100, 115, 130)  
**問題**: 
- TechGo以外のリンクが `https://px.a8.net/sv/e0?affiliateId=...` のような仮のURL
- これらは実際のA8.netのリンクではない

**影響**: クリックしてもアフィリエイト報酬が発生しない

---

#### 🔴 課題3: master_autopilot.py の多くの機能が「シミュレーション」

**場所**: `Affiliate.auto/system/scripts/master_autopilot.py`  
**問題**:
- line 304-310: パフォーマンスデータが乱数で生成（シミュレーション）
- line 182-202: コンテンツ計画がハードコーディング
- line 257-276: SNSスケジュールもハードコーディング
- 実際のAPI（Google Analytics、A8.net、X API）との連携がない

**影響**: 自動化が「見せかけ」で、実際のデータに基づいた改善が行われない

---

### 2.2 重要な課題

#### 🟠 課題4: A8.net審査状況が不明

**憲法より**: A8.netアカウント登録済み、審査待ち  
**問題**: 
- 審査が通っているか不明
- 振込先設定がされているか不明
- 実際のアフィリエイトリンクが発行されているか不明

**影響**: 収益化の根本ができていない可能性

**問題**:
